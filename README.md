# AIナビゲーション PoC - クラフトバンク社向け商談支援ツール

## 📋 プロジェクト概要

クラフトバンク社のCS部門における**ジュニアメンバーのヒアリング品質向上**を目的とした、AI駆動型商談支援ツールのプロトタイプです。

## 🎯 主な機能

### 実装済み機能

- ✅ **音声録音機能**: ブラウザのマイクから会議中の音声を取得
- ✅ **リアルタイム音声レベル表示**: 録音中の音声レベルを可視化
- ✅ **録音データの取得**: MediaRecorder APIを使用した音声データの取得

### 今後実装予定

- ⏳ **文字起こし処理**: Gemini APIを使用した音声のテキスト化
- ⏳ **聞き漏らし判定**: チェックリストと会話内容の突合
- ⏳ **分析レポート生成**: 聞き漏らし項目の可視化

## 🚀 セットアップ

### 前提条件

- Node.js 18.0以上（推奨: 20.x）
- npm 9.0以上

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:5173` を開くとアプリケーションが表示されます。

## 🎤 音声録音機能の使い方

1. ブラウザでアプリケーションを開く
2. 「🎤 録音開始」ボタンをクリック
3. マイクアクセスの許可を確認
4. 録音中は音声レベルが表示されます
5. 「⏹ 録音停止」ボタンで録音を終了

## ⚠️ 注意事項

- マイクへのアクセスにはブラウザの許可が必要です
- HTTPS環境またはlocalhostでのみ動作します（セキュリティ制約）
- 録音データは現在ローカルで処理されます（文字起こし機能は今後実装予定）
